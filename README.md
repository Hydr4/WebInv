WebInv.py
=========

WebInv is a website investigator python script which gets info about a specific website, and here is the list of info and operations which this script does:<br>
1- Status <br>
2- IP<br>
3- Banner Grabber<br>
4- Geolocation IP <br>
5- WHOIS query<br>
